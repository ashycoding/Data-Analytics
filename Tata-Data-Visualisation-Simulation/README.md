# Tata Data Visualisation Simulation  
Empowering Business with Effective Insights  

## Overview
This project was completed as part of the Tata Consultancy Services job simulation on Forage (February 2026).

The objective was to design executive-level dashboards and generate strategic insights to support business decision-making.

## Responsibilities
- Analyzed business datasets
- Designed executive dashboards
- Created visualizations for senior leadership review
- Prepared strategic discussion questions for client meetings

## Tools Used
- Excel
- Data Visualization Techniques
- Dashboard Design Principles

## Outcome
Developed clear and concise visual dashboards enabling leadership to make informed business decisions.
